# Privacy Policy

Last updated: 2026-03-21

Zip Puzzle Studio is published by Appruloft.

## Overview

Zip Puzzle Studio is designed as a general-audience puzzle game. The current project implementation is intended to run locally on the user's device without requiring account creation or sign-in.

## Data Collection

Based on the current implementation in this repository, the app does not intentionally collect, store, or share personal data from users while they play the game.

## Children and Families

The app should be presented as a general-audience puzzle experience and should avoid misleading, harmful, or inappropriate content in gameplay, visuals, store assets, and promotion.

## Permissions

The current game experience does not require sensitive device permissions for core gameplay.

## Data Sharing

Based on the current implementation, no personal data sharing is described for the local-only gameplay experience.

## Data Retention

No player profile or personal data retention is described for the current local-only version.

## Future Changes

If future releases add analytics, advertising SDKs, crash reporting, sign-in, cloud save, payments, external links, or any user data collection, this privacy policy and the Google Play Console Data safety section should be updated before publication.

## Contact

Appruloft should publish a support email address and host this privacy policy at a public HTTPS URL before submitting the app to Google Play.
